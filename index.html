Jadwal Teknisi — 1 Bulan

-
–Belum Diisi MMasuk LLibur SSakit CCuti IIzin PMPiket Malam
✎️ Klik sel status untuk mengubah (– → M → L → S → C → I → PM)
Cara pakai: pilih Bulan Aktif di atas, lalu klik tiap sel status untuk mengisi jadwal teknisi bulan itu (siklus: Belum Diisi → Masuk → Libur → Sakit → Cuti → Izin → Piket Malam). Setiap perubahan otomatis tersimpan dan langsung memperbarui KPI, gauge, tren, serta rekap bulanan. Data tersimpan per bulan sehingga riwayat bulan-bulan sebelumnya tetap bisa dibuka kembali lewat selector bulan atau daftar Riwayat Bulan di bawah. Gunakan filter Sektor untuk melihat jadwal per Sektor Jatinegara 1–4. Daftar teknisi bersifat tetap (satu roster) — gunakan tombol Tambah Teknisi untuk menambah personel baru, dan ikon × di samping nama untuk menghapus. Gunakan tombol Upload Jadwal (Excel) untuk mengganti roster & jadwal bulan aktif langsung dari file Excel/CSV baru (format kolom: NO, NAMA, SEKTOR, UNIT, NO HP, NIK, lalu kolom tanggal 1–31) — pilih dulu Bulan Aktif yang sesuai sebelum upload.

Rekap Kehadiran Teknisi — 1 Bulan

-
Otomatis mengikuti perubahan pada tabel Jadwal Teknisi di atas
No NIK Nama Teknisi Sektor Masuk Libur Sakit Cuti Izin Piket Malam Belum Diisi Total Hari % Kehadiran
Standar kehadiran: target minimal 22 hari kerja/bulan (Masuk + Piket Malam dihitung setara). Warna % Kehadiran — hijau = mencapai target (≥22 hari), merah = belum mencapai target (<22 hari).
